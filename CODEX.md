# Project Instructions

- KubeJS scripts are found in `kubejs/`
    - client scripts in `kubejs/client_scripts/`
    - server scripts in `kubejs/server_scripts/`
    - startup scripts in `kubejs/startup_scripts/`
- Decompiled JS typings are in `.probe/`
- All registered items, blocks, fluids, etc. are found in `.vscode/` snippet files
- Configs are found in `config/`

Always check for the method signature/typing generated by ProbeJS before doing anything script-related. You should never guess an item/block/fluid ID; always check the snippet files.
